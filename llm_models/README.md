# Language Models & Text Representation

This directory serves as the core foundation for understanding how machines comprehend language, building up from basic statistical approaches to advanced vector space representations.

The repository is structured into two main categories:

## 1. Probabilistic Models (`probabilistic_models/`)
Focuses on traditional, statistical, and probability-based approaches to language modeling and sequence prediction.
- Hidden Markov Models (HMM)
- Maximum Entropy Models
- N-Gram Models

## 2. Text Representation (`text_representation/`)
Explores various techniques to convert raw text into numerical features or dense vectors that machine learning algorithms can process.
- **Bag of Words (BoW):** Term frequency-based representations.
- **TF-IDF:** Term Frequency-Inverse Document Frequency.
- **N-Grams:** Capturing local context using sequences of words.
- **Word Embeddings:** Dense vector representations capturing semantic meaning (Word2Vec, GloVe, etc.).

Navigate to each specific subdirectory for detailed implementations and interactive notebooks.
