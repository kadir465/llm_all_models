# Deep Learning Models for NLP

This module focuses on Deep Learning architectures explicitly designed to handle sequential data, such as text, speech, and time-series. The notebooks provide from-scratch understandings or framework-based implementations of core neural network architectures used in Natural Language Processing.

## Contents

- **`recurrent_neural_network.ipynb`**: Covers the fundamentals of Recurrent Neural Networks (RNNs). Explains how RNNs maintain a hidden state to process sequential inputs, along with challenges like the vanishing gradient problem.
- **`lstm.ipynb`**: Deep dive into Long Short-Term Memory (LSTM) networks. Demonstrates how cell states and various gates (input, forget, output) help the network retain long-term dependencies, overcoming the limitations of standard RNNs.

## Prerequisites

To run these models, you will primarily need deep learning frameworks such as TensorFlow or PyTorch. Ensure you have the required libraries installed in your `.venv` environment.
