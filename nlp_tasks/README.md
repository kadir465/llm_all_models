# NLP Tasks

This directory contains various fundamental and advanced Natural Language Processing (NLP) tasks and applications. Each notebook demonstrates how to approach a specific problem, from preprocessing text to implementing state-of-the-art architectures.

## Contents

- **`morphological_analysis.ipynb`**: Demonstrates techniques for analyzing the morphological structure of words, including stemming and lemmatization, tailored for specific languages.
- **`question_answering_bert.ipynb`**: Implementation of a Question Answering (QA) system utilizing the power of bidirectional encoder representations (BERT).
- **`question_answering_gpt.ipynb`**: Generative Question Answering implementation using GPT-based models.
- **`sentiment_analysis.ipynb`**: A comprehensive guide to detecting the emotional tone behind a body of text (positive, negative, neutral).
- **`text_classification.ipynb`**: Techniques for categorizing text into organized groups using various machine learning and deep learning strategies.

## Getting Started

1. Ensure your virtual environment is activated.
2. Install necessary dependencies (e.g., `transformers`, `torch`, `nltk`).
3. Run the cells within each notebook sequentially.
